# 🌌 Demiurge — AI-Powered Worldbuilding Assistant

**Demiurge** — это мощный локальный инструмент для Мастеров Подземелий (Game Masters), писателей и создателей миров. Приложение объединяет в себе вики-систему, интерактивную карту, хронологию событий и продвинутые функции искусственного интеллекта для генерации контента и анализа ваших игровых сессий.

## ✨ Ключевые возможности

*   📖 **Вики-база знаний:** Создавайте и связывайте сущности (NPC, локации, предметы, фракции).
*   🗺️ **Интерактивная карта:** Управляйте галактиками, звездными системами, планетами или регионами фэнтези-мира.
*   ⏳ **Таймлайн (Хронология):** Отслеживайте важные исторические события в удобном формате.
*   📝 **Умный журнал сессий:** Записывайте логи игр. ИИ автоматически проанализирует текст и извлечет новых NPC, локации и события, добавив их в вашу базу!
*   🎲 **Случайные таблицы:** Создавайте таблицы для бросков (лут, энкаунтеры, погода).
*   🤖 **Универсальный ИИ-чат:** Общайтесь с ИИ-ассистентом, который знает весь контекст вашего мира.
*   🎨 **Генерация портретов:** Создавайте аватары для NPC в один клик прямо из их карточек.

## 🚀 Установка и запуск

Проект работает локально на вашем компьютере, сохраняя все данные в локальную базу SQLite.

### Требования
*   [Node.js](https://nodejs.org/) (версия 18 или выше)
*   Git

### Шаги установки

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ВАШ_НИК/demiurge.git
   cd demiurge
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

4. Откройте браузер по адресу: `http://localhost:3000`

## 🧠 Настройка ИИ (Локальные и Облачные нейросети)

Demiurge поддерживает работу как с облачными (OpenAI, Gemini, DeepSeek), так и с **полностью локальными, бесплатными и приватными нейросетями**.

Перейдите в раздел **Настройки (Settings)** в приложении для конфигурации:

### 📝 Текстовые нейросети (LLM)
Вы можете использовать локальные модели через **LM Studio** или **Ollama**, либо подключить облачные API.

**Для локальных моделей (LM Studio / Ollama):**
1. Запустите локальный сервер в LM Studio или Ollama.
2. В настройках Demiurge выберите провайдера: `OpenAI-совместимый`.
3. Укажите Endpoint (например, `http://localhost:1234/v1` для LM Studio или `http://localhost:11434/v1` для Ollama).
4. Укажите название загруженной модели (например, `llama-3-8b-instruct`).
5. API ключ можно оставить пустым или ввести любое слово.

**Для облачных моделей (ChatGPT, DeepSeek, Qwen):**
1. Выберите провайдера: `OpenAI-совместимый`.
2. Укажите Endpoint провайдера (например, `https://api.deepseek.com/v1`).
3. Введите ваш API ключ.

**Для Google Gemini:**
1. Выберите провайдера: `Gemini`.
2. Введите ваш Gemini API Key.

### 🎨 Генерация изображений (Stable Diffusion / DALL-E)
Вы можете генерировать портреты NPC по нажатию одной кнопки (кнопка ✨ в карточке сущности).

**Для локального Stable Diffusion (Automatic1111 / ComfyUI):**
1. Запустите ваш SD с параметром `--api` (для A1111).
2. В настройках выберите провайдера: `Stable Diffusion`.
3. Укажите Endpoint (обычно `http://127.0.0.1:7860`).

**Для OpenAI DALL-E:**
1. Выберите провайдера: `OpenAI`.
2. Введите ваш OpenAI API ключ.

## 💾 Резервное копирование

Все ваши миры хранятся в локальной базе данных SQLite (`database.sqlite`).
Вы можете экспортировать всю вашу вселенную в один JSON-файл прямо из раздела **Настройки** (кнопка "Экспорт") и импортировать её на другом устройстве.

## 🛠 Стек технологий

*   **Frontend:** React, Vite, Tailwind CSS, Lucide Icons
*   **Backend:** Node.js, Express
*   **Database:** SQLite (`better-sqlite3`)
*   **AI Integration:** `@google/genai`, Fetch API для OpenAI-совместимых эндпоинтов
