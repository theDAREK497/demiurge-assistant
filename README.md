# Demiurge Assistant

## О программе

Локальная вики и помошник в ведении своих миров для НРИ, книг и т.д.
В приложении можно настроить локальный LLM для помощи в заполнении или ведение вручную информации.
Приложение предоставляет функционал:
- Wiki с персонажами, местами, предметами, группировками и т.д.
- Упрощенные карты ваших миров
- Таблицы случайных значений по броску и по весу
- Таймлайн важных событий

## Быстрый запуск

Чтобы запустить приложение, вам не нужно ничего устанавливать вручную. Просто выполните эти шаги:

1. Скачайте этот проект и распакуйте его в любую папку.
2. Перейдите в распакованную папку.
3. **Для Windows:** Запустите `start.bat`.
   * *Скрипт сам проверит, есть ли у вас Node.js. Если его нет, он автоматически скачает портативную версию (около 30 МБ) в папку проекта, установит зависимости и запустит сервер. Вам нужно только подождать пару минут.*
4. **Для macOS / Linux:** Откройте терминал в этой папке и выполните команду `bash start.sh` (или дайте файлу права на выполнение `chmod +x start.sh` и запустите `./start.sh`).

После запуска откройте в браузере адрес: **http://localhost:3000**
